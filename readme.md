## アプリケーションの概要
将棋ファンの交流を目的としたサービス

URL http://original-shogi.herokuapp.com/

## アプリケーションの機能一覧
- 認証機能
- 画像投稿機能
- 投稿一覧機能
- 画像へのコメント機能
- コメントに対する参考になった機能
- ユーザ検索機能
- 画像へのタグ機能、タグを使用した検索機能
- 管理者による投稿削除機能


## アプリケーション内で使用している技術一覧
- 開発環境:cloud9
- Laravel:5.5.46
- MYSQL:5.5.62
- PHP:7.2.17
- 画像アップロードサーバ:AmazonS3
- アプリケーションサーバ:Heroku
